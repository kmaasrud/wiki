Ordinary least squares regression

Ordinary least squares (OLS) is a method of estimating the parameters in a #linear regression model.

We have a dataset of n observations (yi​,xi​), where xi​ are the column vectors of length p containing the parameters and yi​ are the scalar responses. We set up the linear function for the i'th case as

yi​=β1​xi1​+β2​xi2​+⋯+βp​xip​+ϵi​,

yi​=xiT​β+ϵi​.

Here β is an unknown p×1 vector and ϵi​ is the error of the model in the i'th case. So for all i cases, we write the total equation on matrix form as

y=Xβ+ϵ,

with X being an n×p matrix, often called the design matrix or matrix of regressors. It's worth noting that - contrary to what one might think - the i'th row of X equals xiT​. Also, the constant term is included in our equation by setting xi1​=1 ∀ i=1,…,n.

Now because of the errors, y=Xβ does not have a unique solution. However, we can find the best fit by finding the vector β^​ that minimizes some sort of cost function#,

β^​=βargmin​S(β).

β^​ will give us an estimate of y, namely y^​=Xβ^​. In the case of OLS, the cost function# we use is the

RSS(β)=i=1∑N​(yi​−y^​i​)2.

Changing into matrix notation again, we get

RSS(β)=(y−y^​)T(y−y^​)=(y−Xβ)T(y−Xβ),

which we can differentiate with respect to β to find the minimum.

∂β∂RSS​=−2XT(y−Xβ).

Assuming full column rank for X, (XTX) is thus positive definite (and importantly, invertible). Setting the first derivative to 0, we get1:

XT(y−Xβ)=0

⇒β^​=(XTX)−1XTy

1

Hastie, T., Tibshirani, R., & Friedman, J. H. (2009). The elements of statistical learning: data mining, inference, and prediction (2nd ed). Springer.
